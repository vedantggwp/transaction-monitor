# Transaction Monitor

A Python-based banking transaction analyzer that generates, analyzes, and flags suspicious transactions.

## Features

- Generates realistic UK banking transaction data
- Categorizes transactions (Groceries, Transport, Entertainment, etc.)
- Flags suspicious transactions (currently: transactions over Â£300)
- Displays transaction summaries with total amounts
